# 🐾 Loving Homes – هونج كونج | فندق الكلاب المميز

موقع ويب احترافي لفندق الكلاب "Loving Homes" في هونج كونج، مصمم بتقنيات عصرية وتجربة مستخدم ممتازة.

## 📋 نظرة عامة

مشروع ويب متكامل مكون من 5 صفحات لتقديم خدمات فندق الكلاب في هونج كونج، مع تصميم عصري وجذاب يدعم جميع الأجهزة.

## 🌟 المميزات الرئيسية

### ✨ التصميم والواجهة
- **تصميم عصري** بألوان دافئة (أخضر طبيعي + بيج + أبيض)
- **متجاوب بالكامل** يعمل على الجوال والتابلت وسطح المكتب
- **دعم RTL** للغة العربية بالكامل
- **رسوميات متحركة** وتأثيرات تفاعلية سلسة

### 📱 الصفحات
1. **الصفحة الرئيسية** - عرض تعريفي مع المميزات وآراء العملاء
2. **الخدمات والمرافق** - تفاصيل جميع الخدمات المتاحة
3. **الحزم والأسعار** - باقات مختلفة بتصميم بطاقات عصرية
4. **تواصل معنا** - نموذج تواصل وخريطة جوجل
5. **من نحن** - قصة الشركة والفريق والشهادات

### 🔧 التقنيات المستخدمة
- **HTML5** - هيكل صفحات دلالي
- **CSS3** - تصميم متقدم مع Grid و Flexbox
- **JavaScript** - تفاعلات حديثة ووظائف ديناميكية
- **Font Awesome** - أيقونات احترافية
- **Google Fonts** - خطوط عربية (Cairo)

## 🚀 التثبيت والتشغيل

### المتطلبات
- متصفح ويب حديث (Chrome, Firefox, Safari, Edge)
